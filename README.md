# Portfolio Project

This repository contains a simple portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main landing page for the portfolio
- `style.css` — styling and layout for the site
- `script.js` — interactive behavior and page enhancements
- `Gloria.jpeg` — profile image for Gloria
- `Afeez.jpeg` — profile image for Afeez
- `.gitattributes` — Git configuration for repository line endings

## About

The portfolio showcases frontend development work and includes a team section, services, skills, projects, and a contact form.

## Usage

1. Open `index.html` in a web browser.
2. Or serve the folder with a local server for best results.
